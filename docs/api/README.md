## API Reference

Public APIs exported by the library entrypoint.

- ExifManager: High-level EXIF operations (read, modify, remove, generate)
- ExifHandler: Browser helper for enriching EXIF (GPS, timestamps)
- Types: Shared TypeScript interfaces
